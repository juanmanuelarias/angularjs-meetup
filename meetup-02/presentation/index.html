Workflow & Automation

Yeoman

“Yeoman 1.0 is more than just a tool. It's a workflow; a collection of tools and best practices working in harmony to make developing for the web even better.”

¿Por que Yeoman?

  • Nos permite crear Web Applications de forma rápida, evitando tareas manuales
  • Utiliza herramientas "standards" de la comunidad open-source
  • Y además, es extensible!

El Workflow se compone de:

  1. Yeoman: scaffolding de una nueva aplicación
  2. Grunt: utilizado para el Build, Preview y Test de nuestro proyecto
  3. Bower: manejador de dependencias
