¿Que es AngularJS?

Framework JavaScript Open Source

Desarrollado por Google

Aplica el patrón MVC


<h1>{{persona.nombre}}</h1>
                    

<script>
    $scope.persona = { nombre: 'Juan' };
</script>

¿Cuales son las Ventajas?

Aplica Dependency Injection

Facilita la construcción de SPA con el uso de rutas y bindings

Permite desacoplar la manipulación del DOM de la lógica

Extiende el vocabulario HTML

Facilidad de realizar Testing Unitario y E2E

Los objetos no necesitan una estructura especifica de Getters y Setters

<script>
    function ItemsCtrl($scope) {
        $scope.count = 0;
        $scope.items = [];
        $scope.addItem = function () {
            $scope.count++;
            $scope.items.push('Nuevo Item ' + $scope.count);
        }
    }
</script>

	$scope.tasks = [ 
	    { title: 'Fix Bug #3457', status: 'Done' },

	]
	$scope.addTask = function (task) {

Services

  • Objetos con lógica de la aplicación
  • Suele comunicarse con el Backend
  • Objetos reutilizables

Services


myApp.controller('TaskCtrl', function ($scope, taskData) {
    $scope.tasks = taskData.tasks;
    $scope.add = function (task) {
        taskData.add(task);
    }
});
                    

myApp.factory('taskData', function () {
    return {
        tasks : [
            { title: 'Fix bug #3445', status: 'done', ... },
            { title: 'Fix bug #3446', status: 'done', ... },
            { title: 'Add new Feature', status: 'pending', ... }],
        add: function (task) {
            this.tasks.push(task);
        }
    }
});

Routing

  • Permite hacer SPA
  • Navegación con History
  • Utiliza el # en las rutas o nada si es HTML5

<body>
    <ng-view></ng-view>
</body>
                    

var app = angular.module('myApp', [])
    .config(function($routeProvider){
        $routeProvider.when('/tasks',{
            templateUrl: 'views/tasks.html',
            controller: 'TaskCtrl'
        }
    });
En modo HTML5, el server tiene que responder a todas las rutas para que angular pueda cargarla y operar sobre ella.
